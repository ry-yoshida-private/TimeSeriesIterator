# utils

Utility enums used by the time series iterator package.

## Components

| Component | Description |
|-----------|-------------|
| [index_base.py](./index_base.py) | Enum defining the base index for time series IDs (`ZERO` / `ONE`) |
| [media_type.py](./media_type.py) | Enum defining supported media types (`IMAGE` / `VIDEO` / `AUDIO`) |
